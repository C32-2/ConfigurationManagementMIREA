## Задание 5

Написать программу для регистрации пользовательской команды (правильные права доступа и копирование в /usr/local/bin).

```bash
#!/bin/bash 

d="/usr/local/bin/$(basename "$file" .py)" 
file=$1 

if [ ! -f $file ]; then 
    echo "File $file not found" 
    exit 1 
fi 

cp $file $d 
chmod 755 $d 
```
